What is the difference between map, filter, forEach and find method?

Answer: Map array method loop through an array and return a new array.
Filter array method creates a copy of the array and return the values which pass by the given conditon.
forEach array method just loop through the given array and dosen't return any value.
Find array method only return the first element which pass the condition, if there is no value then it returns undefined.

What is eventLoop, how it works?

Answer: The Event Loop maintain the call stack and the callback queue. When the call stack gets empty the the event loop will take the first event
from the queue and push that into the call stack and run it.
